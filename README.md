# SnapVanish 📸✨

SnapVanish is a lightweight desktop tool that lets you capture a screenshot and use it instantly — without saving it permanently.

## 🚀 Features

* Capture screenshots using a shortcut
* No permanent storage (privacy-first)
* Runs in the background
* Fast and simple workflow

## 🛠 Tech Stack

* Electron
* Node.js

## ⚡ How it Works

1. Press the shortcut (Ctrl + alt +X)
2. Screenshot is captured in memory
3. Used for processing/search
4. Automatically discarded

## 🎯 Goal

To build a privacy-focused screenshot tool that leaves no trace on your system.
